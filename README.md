# Company Research

Standalone Drupal module repo for automated company and career-site research.

## What this module does

- discovers company websites and careers pages
- identifies likely ATS platforms
- analyzes authentication patterns for downstream automation planning

## Key dependencies

- `ai_conversation`
- Drupal node/field/user/system services

## Repo note

This repository is the source of truth for the `company_research` module and should be maintained independently from the HQ repo.
